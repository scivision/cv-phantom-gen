==============
cv-phantom-gen
==============

Computer Vision phantom generation, particularly useful for simulated images of the aurora borealis. Usable from Python, Octave, or Matlab.

Install
=======
::

    python setup.py develop

Usage
=====
::
    
    ./AuroraPhantom.py
